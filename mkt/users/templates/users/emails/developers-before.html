In the near future, Firefox Marketplace will retire its current user sign-in system (Persona) in favor of a far more robust user account system we’re calling Firefox Accounts. For more information, please see the Firefox Accounts FAQ.

When the transition from Persona to Firefox Accounts is ready, you’ll receive an email with an activation link.
